# Streamlit
It is toole that crates app without Frontend adn Backend imprementation.

## How to use it?

### Install
```
conda install -c conda-forge streamlit
conda install pipenv
```

### Start Hello Demo

```
streamlit hello
```

#### Run app

```
streamlit run main.py
```

## Docker

```
$ docker build -t myapp .
$ docker run -p 8501:8501 myapp
```
